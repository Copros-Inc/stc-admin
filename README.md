# STC 管理画面

下関トレーニングセンター（STC）の総合管理システムです。

## 🎯 機能

### 📅 スケジュール管理
- 講習予定の追加・編集・削除
- 月別表示
- ステータス管理（空席あり / 満員 / 中止）

### 📋 予約受付管理
- 申し込まれた予約情報の確認
- 予約の削除機能
- リアルタイム更新

### 📄 書類作成管理
- 修了証の発行（準備中）
- 領収書の発行（準備中）
- 請求書の作成（準備中）
